# attractors

**Reproducing Fast Weights results**

      cd repro_fw
      python retrieval.py
      python retrieval.py lstm_vs_fw 8

This will reproduce the results in section 4.1 of the paper with 30 hidden units and learned 64 dimensional embeddings.
